How to use the app

  1. Start a new session or continue an existing one.
  2. Select one or multiple image files.
  3. Choose the LLM model (Mistral or FLAN-T5).
  4. Click Upload to process the images.
  5. Results appear in the chat area above.
  6. Invalid or unsupported files are skipped with a warning.
